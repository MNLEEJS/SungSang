
# 개발자 환경 구축

1. JDK와 JRE는 무엇인가요?

○ . JDK 란 Java Development Kit의 약자

- Java 프로그램의 실행뿐만 아니라 소스코드의 작성과 컴파일 테스트, 패키징과 같은 빌드 과정까지 수행할수 있게 도와줌

- Java 프로그램을 개발하려는 개발자에게 필요한 환경을 제공

- JDK는 JRE의 모든 기능을 포함하며 작성, 컴파일, 디버그하고 문서화할 수 있는 도구 제공



○ . JRE 란 Java Runtime Environment의 약자

- JRE는 java 프로그램을 실행시키는 데 필요한 라이브러리와 java프로그램을 실행할 수 있게 해주는 JVM을 제공

- JAVA프로그램을 실행시키는데만 사용되는 패키지




2. 설치 과정과시스템환경 변수 설정 방법을 문서로 작성해보세요.

■ 최초 구글에서 JDK 1.8 검색 하여 제일 상단에 뜨는 Oracle 사이트로 접속

■ 내가 사용하는 pc 버전에 맞게 JDK 다운로드

■ 다운로드 받은 파일 실행

( 모르면 내컴퓨터 속성에서 체크 가능 )           예) Window bit x64  ,   mac os ( 애플 )


# 어느 폴더에서든 JAVA를 실행 할수 있게 하는 방법 #



1. 윈도우 찾기에서 " 시스템 환경 변수 편집" 검색

2. 환경 변수 클릭

3. "시스템 변수" 칸에서 " 새로 만들기 " 클릭

4. 변수 이름 칸에 " JAVA_HOME" 작성

5. 디렉터리 찾아보기 클릭

6. 내가 설치한 JAVA JDK 폴더 누르고 확인

7. 환경 변수에서 변수 " Path " 더블클릭

8. "새로 만들기" 클릭

9. " %JAVA_HOME%\bin " 작성 후 확인
