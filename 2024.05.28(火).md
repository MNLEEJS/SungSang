## JDK 란?
-  자바 개발에 필요한 도구들의 집합체
-  주요 구성 요소
	- javac : 자바 컴파일러로, 자바 소스 코드를 바이트 코드로 변환
	- java : 자바 애플리케이션을 실행하는 도구
	- javadoc : 자바 소스 코드에서 API 문서를 생성하는 도구
	- jdb : 자바 디버거 로 자바 프로그램을 디버깅 하는데 사용
